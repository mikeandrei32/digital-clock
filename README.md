# Digital Clock

A simple and elegant digital clock built with React that displays the current time in a 12-hour format with AM/PM notation. The clock updates every second and features a modern glassmorphism-style interface with a fullscreen background image.

## Features

- Real-time clock updates every second
- 12-hour time format with AM/PM
- Automatic leading zeros for single-digit values
- Responsive design
- Clean and modern UI

## Technologies Used

- React
- JavaScript
- CSS
